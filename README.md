# Simple Interest Calculator

This project is a simple bash script that calculates **simple interest**
based on user input.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Inputs
- Principal amount
- Rate of interest
- Time period (in years)

## Output
- Calculated simple interest

## Usage
Run the script in a terminal and follow the prompts to enter the required values.
